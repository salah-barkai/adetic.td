# ADETIC Website

Site officiel de l'Agence de Développement des Technologies de l'Information et de la Communication du Tchad.

## 🚀 Installation

```bash
npm install
```

## 📦 Dépendances

- React 18.3.1
- React DOM 18.3.1
- Vite 5.4.0
- @vitejs/plugin-react 4.3.1

## 🛠️ Scripts

- `npm run dev` - Lance le serveur de développement
- `npm run build` - Crée un build de production
- `npm run preview` - Prévisualise le build de production

## 📝 Structure du projet

```
adetic-website/
├── index.html          # Point d'entrée HTML
├── package.json        # Dépendances et scripts
├── vite.config.js      # Configuration Vite
└── src/
    ├── main.jsx        # Point d'entrée React
    └── App.jsx         # Composant principal
```

## 🎨 Fonctionnalités

- Navigation fixe avec effet de scroll
- Section Hero avec animations
- Section Actualités avec sélection interactive
- Section Missions avec cartes interactives
- Section Direction Générale
- Section Activités
- Formulaire de contact
- Footer avec liens

## 🌐 Déploiement

Le projet est configuré pour être déployé sur n'importe quel hébergeur statique (Vercel, Netlify, etc.).
